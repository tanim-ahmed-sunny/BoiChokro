Read. Reuse. Reduce Waste.

Boichokro is a modern, eco-friendly web platform designed to give books a second life. By facilitating book swaps, sales, and donations, we aim to build a circular economy for knowledge while significantly reducing the environmental footprint of the publishing industry.

---

 Our Mission
Every reused book saves paper, water, and energy. Boichokro (meaning "Book Circle") connects readers to ensure that no book ends up in a landfill when it could be in someone's hands.

---
Key Features

- Versatile Marketplace: Buy, swap, or donate used books with ease.
- Library Management: Integrated tools for local libraries to manage their collections and lending history.
- Vibrant Community: A dedicated "Lounge" for readers to discuss books, share reviews, and participate in weekly prompts.
- Environmental Impact Tracking: Real-time metrics showing how many trees have been saved and CO2 emissions reduced through the community's collective actions.
- Smart Notifications: Stay updated on swap requests, wishlist matches, and community interactions.
- Responsive & Premium UI: A clean, "warm illustrated" design aesthetic that feels calm and professional across all devices.

---

 Technology Stack

- Backend: PHP 8.x
- Database: MySQL / MariaDB
- Frontend: HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- Frameworks/Libraries: 
  - Bootstrap 5.3 (Layout & Components)
  - Bootstrap Icons
  - Google Fonts (Inter & Serif family)
- Architecture: Role-based access control (Admin, Library, User).

 Getting Started

Prerequisites
- PHP >= 7.4
- MySQL / MariaDB
- A local server environment like **XAMPP**, **WAMP**, or **Laragon**.

Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/tanim-ahmed-sunny/BoiChokro
   cd boichokro
   ```
2. Setup Database:
   - Create a database named `boichokro` in your MySQL server.
   - Import the SQL files from the `migrations/` or `database/` directory (if available).
3. Configuration:
   - Update `backend/includes/db.php` with your local database credentials.
4. Run:
   - Move the project directory to your `htdocs` or `www` folder.
   - Open your browser and navigate to `http://localhost/boi`.
